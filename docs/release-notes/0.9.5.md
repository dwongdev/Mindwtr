# Mindwtr 0.9.5

## Draft Highlights

- Sync reliability improves with persisted project sequential scope across CloudKit, desktop SQLite, and mobile/desktop sync paths.
- Search and task editing now treat task location as a first-class field, including SQLite full-text search and `location:` / `where:` operators.
- Mobile sync no-change checks avoid stale fast-cache fingerprints and force a full cycle after attachment pre-sync mutates local data.
- Calendar behavior is more consistent: desktop shows date-only start dates as all-day scheduled tasks, event-to-task creation preserves event locations, mobile calendar push writes task locations, and reference tasks are removed from pushed calendars.
- Mobile markdown code-block copy now uses the supported Expo clipboard API.
