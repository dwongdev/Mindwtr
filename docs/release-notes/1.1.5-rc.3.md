# Mindwtr 1.1.5-rc.3

Use this prerelease to test the changes since `1.1.5-rc.2`, especially project sorting, sync safety, mobile performance, and Android reminder scheduling. Install it through tester channels.

## RC Testing Focus

- Upgrade a large mobile library from `1.1.5-rc.2`, make several quick edits, background the app, and sync while the app is opening. Saves should stay responsive, the safety backup should flush on background, and a completion made during startup sync should not revert or lose its next recurrence.
- Pick different task sorts in several projects, restart, and sync between devices. Each project should remember its own sort; manual drag order should still apply when that project uses the default sort.
- On iOS and macOS, sync Focus stars, reminder suppression, future-recurrence previews, sequential-project settings, project Focus, and the new project sort. Values set on iPhone or iPad should remain intact on the Mac.
- On Android, exercise daily digests, weekly reviews, and repeating reminders across reboot, battery optimization, and a daylight-saving change where practical. Each occurrence should fire at its configured wall-clock time and schedule the next one exactly once.
- In a section-scoped sequential project, finish the section's last next action. The follow-up prompt should stay in that section and should not offer to complete the still-active project.
- Test mobile bulk actions in Review, Contexts, and Archive, multi-line quick capture, checklist return-key entry, Time Spent badges, and the new flat outlined task cards. Errors should not produce false success toasts.
- Test WebDAV after an app restart without retyping the password, then force a failing request. The shared log should identify the method, final URL, status, and server response. Invalid short self-hosted tokens should be rejected before saving or server startup.
- On desktop, try Condensed density, bare `1`-`9` area switching, Completed/Waiting/Someday filters, date shortcuts inside the calendar, focused-task stars, and muted queued tasks in sequential projects. On Windows, verify normal links, media, sync, and updates with the hardened WebView2 network flags.

## Highlights

- Remembers task sorting per project and syncs it across devices.
- Prevents stale startup-sync snapshots and SQLite snapshot saves from overwriting or deleting newer local rows.
- Speeds up large mobile libraries by throttling full JSON safety backups while preserving an immediate background flush.
- Re-arms Android repeating reminders as exact alarms so daily and weekly schedules resist battery and reboot drift.
- Preserves iOS-written CloudKit booleans on macOS and improves WebDAV, manual-sync, and self-hosted-token diagnostics.
- Adds section-scoped next-action prompts, clearer Focus grouping, consistent bulk actions, and flatter mobile task cards.
- Adds Condensed desktop density, direct area number shortcuts, list filters, calendar-contained quick dates, and clearer sequential/focused task indicators.
- Improves mobile checklist entry, multi-line capture, task-editor wording, Time Spent visibility, and unreachable-server guidance.
