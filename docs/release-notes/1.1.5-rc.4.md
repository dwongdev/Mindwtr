# Mindwtr 1.1.5-rc.4

This prerelease focuses on changes since rc.3, especially Linux system calendars, storage repair, quick-add parsing, Android editing and widgets, and private-network sync.

## RC Testing Focus

- **Linux system calendars:** Test Evolution Data Server calendar read/write, selecting a push calendar, Flatpak permissions, and startup without EDS installed.
- **Storage and MCP:** Exercise search after database repair, concurrent desktop/MCP writes, and task writes that return the persisted result with any repair warning.
- **Quick add:** Test trailing dates with title cleanup disabled, smart or mixed quotes, and European dot dates such as `26.06.`.
- **Android:** Edit long task descriptions above the keyboard, verify widget due labels, and confirm widgets refresh after background changes without excessive updates.
- **Sync:** Test custom HTTP hosts with the insecure-HTTP override both enabled and disabled, wrong endpoint errors, and the macOS local-network prompt.
- **Desktop:** Test Standard-list shortcuts, token badges, long translated list titles, new-task scrolling, and Windows exit/sync behavior.
- **AI:** Verify the refreshed model catalog and that provider errors are shown instead of generic failures.

## Highlights

- Adds Linux system-calendar integration through Evolution Data Server.
- Makes full-text-search repair atomic and restores its triggers after database repair.
- Expands quick-add date and quote parsing across desktop, mobile, and Shortcuts.
- Improves Android task editing, widget due labels, and widget refresh throttling.
- Improves private-network sync overrides and endpoint diagnostics.
- Adds desktop task shortcuts, clearer quick-add tokens, and current AI model options.
